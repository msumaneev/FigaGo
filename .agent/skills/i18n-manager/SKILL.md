# 🌐 i18n-manager — Управление локализацией FigaGo

## Когда использовать
- Добавление **новой надписи, кнопки, диалога, toast, TTS-фразы**
- Исправление или уточнение **существующего перевода**
- Аудит — проверка, что **все ключи** присутствуют во всех языковых файлах

## Обязательный процесс при добавлении строки

### Шаг 1. Контекст
Перед переводом ВСЕГДА определить:
- **Где используется** — экран, элемент UI (кнопка, заголовок, описание, toast, TTS)
- **Ограничения** — максимальная длина (кнопки ~15 символов, заголовки ~25, описания ~60)
- **Тон** — формальный (диалоги), дружественный (TTS), нейтральный (метки)

### Шаг 2. Мастер-запись
Добавить строку в мастер-таблицу переводов:
```
Артефакт: translation_master.md (в brain текущей сессии)
```

Формат записи:
```
| `ключ` | Контекст использования | ru | en | tr | es | fr | de | uk | ar | ka | ja | ko | zh-CN | zh-TW |
```

### Шаг 3. Обновление ВСЕХ strings.xml

Поддерживаемые языки и папки:

| Код | Язык | Папка ресурсов |
|-----|------|----------------|
| ru | Русский (базовый) | `values/` |
| en | English | `values-en/` |
| tr | Türkçe | `values-tr/` |
| es | Español | `values-es/` |
| fr | Français | `values-fr/` |
| de | Deutsch | `values-de/` |
| uk | Українська | `values-uk/` |
| ar | العربية | `values-ar/` |
| ka | ქართული | `values-ka/` |
| ja | 日本語 | `values-ja/` |
| ko | 한국어 | `values-ko/` |
| zh-CN | 简体中文 | `values-zh-rCN/` |
| zh-TW | 繁體中文 | `values-zh-rTW/` |

**КРИТИЧНО:** При добавлении ключа в `values/strings.xml` — ОБЯЗАТЕЛЬНО добавить его во ВСЕ 12 остальных файлов. Пропуск недопустим.

### Шаг 4. Генерация через скрипт
Для массового обновления использовать скрипт:
```
python c:\FigaGo\.agent\skills\i18n-manager\scripts\generate_strings.py
```

## Правила качества переводов

### DO ✅
- **Контекстные переводы** — «Начать поездку» для коляски → tr: `Sürüşü Başlat` (не `Harekete Geç`)
- **Голосовые команды** — естественная речь: «Я поехал» → en: `Let's go` (не `I started`)
- **Единицы измерения** — локальные стандарты: tr: `km/s` (не `km/h`), ru: `км/ч`
- **Длина текста** — учитывать ширину кнопок (немецкий обычно на 30% длиннее)
- **Эндонимы** — названия языков в выборе (`lang_*`) — ВСЕГДА самоназвание

### DON'T ❌
- Машинный дословный перевод без контекста
- Копирование английского варианта во все языки
- Забывать про RTL-языки (арабский) — тестировать направление
- Использовать кавычки `"` в XML без экранирования — использовать `«»` или `\"`
- Апострофы `'` в XML — экранировать как `\'` или заменять на другие кавычки

## Структура контекста приложения

FigaGo — GPS-трекер для людей на инвалидных колясках (электрических и ручных).

Основные экраны:
- **Dashboard** — карта + метрики (пробег, скорость, время) + управление треком
- **History** — список прошлых поездок по дням
- **DayDetail** — статистика конкретного дня
- **Settings** — GPS, TTS, батарея, единицы, язык, голосовой помощник
- **ProfilePassport** — «паспорт» коляски (тип, скорость, иконка, батарея)
- **Diagnostics** — проверка системных разрешений
- **Permissions** — первый запуск, диалоги разрешений

Специфическая терминология:
- **LED / Лампочки** — индикаторы заряда на пульте электроколяски
- **Транспорт** — режим когда пользователь едет в автобусе/машине (скорость выше порога)
- **Точка невозврата (PONR)** — предупреждение что заряда может не хватить на обратный путь
- **TTS** — Text-to-Speech, голосовые оповещения о пробеге

## Аудит

Для проверки полноты переводов:
```powershell
python c:\FigaGo\.agent\skills\i18n-manager\scripts\audit_strings.py
```
Выводит список отсутствующих ключей по каждому языку.
