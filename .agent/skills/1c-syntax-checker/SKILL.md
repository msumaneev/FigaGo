# Валидатор синтаксиса 1С (BSL Language Server)

Этот навык используется для автоматической проверки исходного кода 1С (папок с выгруженными обработками или конфигурацией) на наличие синтаксических ошибок и code-smells с помощью BSL Language Server.

## Инструкция по использованию

Если пользователь просит **"проверить синтаксис"**, **"сделать ревью кода 1С"** или **"найти ошибки в обработке/коде"**:

1. Убедись, что исходный код (выгруженный в формате файлов 1С) доступен в какой-либо директории (например, в рабочей директории выгрузки обработки).
2. Запусти скрипт проверки синтаксиса:
   ```bash
   python ".agent/skills/1c-syntax-checker/scripts/checker.py" --src-dir "<ПолныйПутьКИсходникам>"
   ```
3. Скрипт скачает (при первом запуске) актуальную версию `bsl-language-server_win.zip`, распакует её и выполнит анализ.
4. Проанализируй вывод скрипта в консоле (stdout):
   - Если ошибок нет, сообщи пользователю об успешной проверке.
   - Если найдены ошибки, предоставь пользователю сводку с указанием файлов и строк, где возникли проблемы, и предложи пути их исправления.
