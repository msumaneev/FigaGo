# Справочник синтаксиса 1С (BSL)

Локальная офлайн-справка по встроенным функциям платформы 1С:Предприятие.
Работает без подключения к серверу — все данные в JSON-файле.

## Инструкция

Если пользователь спрашивает **«как работает функция X»**, **«какие параметры у Y»**, **«синтаксис Z»**:

1. Прочитай файл `.agent/skills/1c-syntax-helper/data/bsl_help.json`
2. Найди объект, у которого `name` или `name_en` совпадает с запрошенной функцией (регистронезависимый поиск)
3. Выдай пользователю:
   - **Синтаксис** (`syntax`)
   - **Параметры** (`params` — массив с описаниями)
   - **Возвращаемое значение** (`returns`)
   - **Пример** (`example`)

## Пример ответа

```
### СтрНайти (StrFind)

**Синтаксис:** СтрНайти(Строка, ПодстрокаПоиска, НаправлениеПоиска, НачальнаяПозиция, НомерВхождения)

**Параметры:**
- Строка (обязательный) — исходная строка
- ПодстрокаПоиска (обязательный) — искомая подстрока
...

**Возвращает:** Число — позиция первого символа найденной подстроки

**Пример:**
Позиция = СтрНайти("Привет, мир!", "мир"); // 9
```

## Расширение базы

Файл `data/bsl_help.json` можно расширять вручную, добавляя новые объекты в массив.
Формат каждой записи:

```json
{
  "name": "ИмяФункции",
  "name_en": "FunctionName",
  "syntax": "ИмяФункции(Параметр1, Параметр2)",
  "params": [
    {"name": "Параметр1", "required": true, "description": "Описание"},
    {"name": "Параметр2", "required": false, "description": "Описание"}
  ],
  "returns": "Описание возвращаемого значения",
  "example": "Результат = ИмяФункции(\"тест\");"
}
```
