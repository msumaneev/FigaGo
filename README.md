# FigaGo

**FigaGo** — это специализированное Android-приложение для GPS-трекинга маршрутов, созданное с фокусом на пользователей электрических инвалидных колясок. Приложение не просто пишет трек, но и ведет умную аналитику расхода заряда батареи, предотвращая ситуации внезапной разрядки вдали от дома.

## Основные возможности

- **Точный GPS-трекер с авто-паузой:** Работает в фоне (Foreground Service), автоматически определяет остановки и строит точный маршрут на карте (с использованием Google Maps). Поддерживает автозакрытие истории текущего дня.
- **Предиктивная аналитика батареи:** Интерактивные индикаторы предсказывают оставшийся запас хода (в километрах). Приложение использует данные предыдущих поездок для адаптивного расчета реального пробега на каждом из 5 делений заряда коляски.
- **Голосовое управление (Hands-Free):** Глубокая интеграция с Google Assistant (через Routines и App Links). Вы можете запускать приложение или проверять статус без рук с помощью кастомных команд: _«Окей, Гугл, я поехал»_, _«Какой пробег?»_ или _«Сделай паузу»_.
- **TTS Оповещения (Text-To-Speech):** Встроенный диктор регулярно информирует о пройденном расстоянии или прошедшем времени, помогая следить за планом поездки без необходимости смотреть на экран.
- **Инклюзивный интерфейс:** Полностью обновленный современный UI на базе Jetpack Compose. Экран настроек использует крупные шрифты, читабельные подсказки и специальные "барабаны" (Wheel Pickers) вместо классической клавиатуры для удобства пользователей с ограничениями моторики.

## Технологический стек

Приложение построено по стандартам **Clean Architecture (MVVM)** и использует современные инструменты экосистемы Android:

- **Основа:** Kotlin, Android SDK (Min 26, Target 35)
- **UI Слои:** Jetpack Compose, Material 3
- **Работа со стейтом:** Flow / StateFlow, ViewModel
- **База данных и кэш:** Room (SQLite) для истории маршрутов, DataStore для настроек
- **Внедрение зависимостей:** Dagger Hilt
- **Геолокация и Карты:** Google Play Services Location, Google Maps Compose
- **Интеграции:** Android App Links, TextToSpeech

## Структура проекта

```
com.figago
├── data/          # DAO (Room), Имплементация репозиториев (Repository), Мапперы
├── di/            # Hilt модули для провайдинга зависимостей
├── domain/        # Бизнес-модели, Интерфейсы репозиториев, Use-cases

└── voice/         # Обработка интеграций App Links / Voice Routines
```

## Сборка и запуск

1. **Окружение:** Установите Android Studio (настоятельно рекомендуется последняя версия) и актуальный Android SDK.
2. **Ключ API:** Для работы карт вам понадобится Google Maps API Key. Поместите его в файл `local.properties` в корне проекта:
   ```properties
   MAPS_API_KEY=AIzaSy...ваш_ключ...
   ```
3. **Синхронизация:** Выполните _Sync Project with Gradle Files_.
4. **Запуск:** Запустите сборку на реальном устройстве (предпочтительнее для тестов GPS).
