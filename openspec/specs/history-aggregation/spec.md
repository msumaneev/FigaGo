## ADDED Requirements

### Requirement: Show All Profiles for a Day
Пользователь должен иметь возможность посмотреть единый сводный трек за день, даже если он пересаживался с одной коляски на другую.

#### Scenario: User toggles "All Profiles" view
- **WHEN** На экране Истории пользователь переключает Switch или RadioButton в режим "Все коляски"
- **THEN** Приложение подгружает агрегированный список сессий со всех профилей (сгруппированный по датам) и отображает суммарную дистанцию, а на карте строит мульти-Polylines.

### Requirement: Fix track_segment Profile Binding
- **WHEN** Начинается новая сессия/отрезок
- **THEN** Приложение должно сохранять в базу корректный `profile_id`, чтобы база данных была согласованной.
