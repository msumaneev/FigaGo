## MODIFIED Requirements

### Requirement: Global Carousel Placement
Система SHALL отображать карусель профилей над контентом всех основных вкладок
(Старт, История, Карта, Настройки), но НЕ на вложенных экранах (детали дня, паспорт профиля).

#### Scenario: Carousel visibility on main tabs
- **WHEN** пользователь находится на вкладке «История»
- **THEN** карусель профилей отображается сверху
- **AND** содержит плитки всех созданных колясок + кнопку «Добавить»

#### Scenario: Carousel hidden on nested screens
- **WHEN** пользователь открывает экран «Детали дня»
- **THEN** карусель профилей НЕ отображается

#### Scenario: Instant auto-creation from carousel
- **WHEN** пользователь нажимает плитку "Добавить" в карусели
- **THEN** система создает новый профиль коляски в БД с инкрементальным именем (Новая коляска)
- **AND** происходит редирект на вкладку «Настройки» для редактирования этого нового профиля
