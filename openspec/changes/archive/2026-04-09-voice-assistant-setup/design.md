## Context

Нам нужно упростить ручную настройку Google Assistant Routines (Процедур) для пользователя. Полная автоматизация заблокирована Google-ом (отсутствуют нужные API), поэтому единственным решением остается быстрый переход на страницу настроек Ассистента.

## Goals / Non-Goals

**Goals:**
- Дать пользователю кнопку "Перейти к настройкам Ассистента" в приложении FigaGo на экране настроек.
- Сэкономить клики пользователя при поиске настроек в Android.

**Non-Goals:**
- Использовать закрытые или недокументированные API Google Assistant для прямого добавления процедур.

## Decisions

- **Launch URL Intent**: Приложение будет использовать неявный (implicit) `Intent` с открытием URL `https://assistant.google.com/settings/routines`. Это стандартный URL, который перехватывается приложением Google/Google Assistant и ведет прямо в нужный раздел. Если этот URL не поддерживается на старых устройствах, он откроется в браузере, и браузер также перенаправит пользователя. Можно также использовать `com.google.android.googlequicksearchbox` для fallback.

## Risks / Trade-offs

- Риск, что на устройствах без Google Services (например, некоторые Huawei) Intent вызовет `ActivityNotFoundException` → Нужно обернуть вызов в `try-catch` и показать `Toast` с предупреждением, если перейти невозможно.
