## ADDED Requirements

### Requirement: Point of No Return Alert
Система SHALL произносить голосовое предупреждение однократно за сессию, когда текущий пробег достигает точки невозврата.

#### Scenario: Triggering point of no return warning
- **WHEN** текущая пройденная дистанция превышает Точку Невозврата минус предупреждающий зазор
- **THEN** система произносит сообщение "Точка невозврата через X километров" и блокирует повтор для текущей сессии

### Requirement: Distance TTS Rounding
Система SHALL округлять озвучиваемые пройденные километры по математическим правилам, а остаточные усекать (floor).

#### Scenario: Rounding up passed distance
- **WHEN** пройденная дистанция 5.6 км
- **THEN** система округляет её до 6 км для озвучки

#### Scenario: Truncating remaining distance
- **WHEN** расчетный остаток 23.7 км
- **THEN** система усекает его до 23 км для озвучки
